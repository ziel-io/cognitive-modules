# cogn

Cognitive Modules CLI - 结构化 AI 任务执行工具。

这是 `cognitive-modules-cli` 的短名别名包。

## 一行代码使用

```bash
# 零安装运行
npx cogn run code-reviewer --args "def login(u,p): pass"

# 列出模块
npx cogn list

# 查看帮助
npx cogn --help
```

## 全局安装

```bash
npm install -g cogn

# 然后直接使用
cogn run code-reviewer --args "..."
```
