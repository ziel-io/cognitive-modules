# Cognitive Modules Enhancement Proposals (CMEPs)

This directory contains all CMEPs (Cognitive Modules Enhancement Proposals).

## What is a CMEP?

A CMEP is a design document that proposes a change to the Cognitive Modules specification. See [CMEP-PROCESS.md](../CMEP-PROCESS.md) for the full process.

## Creating a CMEP

1. Copy [TEMPLATE.md](TEMPLATE.md) to `CMEP-XXXX.md`
2. Fill in all sections
3. Open a Pull Request
4. Address feedback
5. Get approval

## CMEP Lifecycle

```
Draft → Review → Accepted → Final → Active
         ↓
      Rejected
```

See [CMEP-PROCESS.md](../CMEP-PROCESS.md) for details.
